# LNL/LZS Conceptual Visualizations

This directory contains conceptual 3D visualizations of the Lange Number Line (LNL/LZS) Collatz research project.

The images illustrate structural ideas, trajectories, mountains, turning points and number landscapes. They are explanatory visualizations and do not constitute mathematical proofs or numerical verification results.
